position 프로퍼티
    + 요소(태그)의 위치를 지정한다.
    + top, bottom, left, right 프로퍼티와 함께 사용하여 위치를 지정한다.(벽에서 얼마나 멀어질건가를 결정하는것)

position 프로퍼티 값
    + static(기본위치)
        position프로퍼티의 기본값이다
        요소의 배치 순서에 따라 위에서 아래로, 왼쪽에서 오른쪽으로 순서에 따라 배치된다.
        좌표 프로퍼티(top, bottom, left,right)를 사용 할수없다.

    + relative(상대위치)
        기본위치 (static으로 지정했을 때의 위치)를 기준으로 좌표 프로퍼티를 이용해서
        위치를 이동시킨다.
        
    + absolute
        + 부모 또는 조상요소의 positiondl 프로퍼티가 static이면
          페이지의 원점을 기준으로 좌표값만큼 이동해서 위치한다.
        + 부모 또는 가장 가까운 조상(static은 제외)을 기준으로 지정된 좌표값만큼 이동한다.
          즉, relative, absolute, fixed 프로퍼티가 선언된 부모 또는 조상 요소를 기준으로 
          위차가 결정된다.

    + fixed (화면이 이동되도 그위치가 계속 유지되는것)
        + 부모요소와 관계없이 브라우저의 viewport을 기준으로 과표값만큼 위치를 이동시킨다.
        + 스크롤이 되더라도 화면에서 사라지지 않고 항상 같은 곳에 위치한다
        + fixed 프로퍼티 선언시, block요소의 width는 inline요소와 같이
        content에 맞게 변화되므로 적절한 width를 지정해야한다.

z-index 프로퍼티(특정요소가 앞으로 나오거나 뒤로 오게할수 잇는것)
        + z-index 프로퍼티 값이 클수록 화면 전면에 출력된다.
        + position 프로퍼티가 staic이 아닌 요소에만 적용된다.
