셀렉터(Selector, 선택자)
    + style을 적용하고자 하는 HTML 요소를 지정하는 것이다.

선택자의 종류
    + 전체 선택자 
        패턴       설명
        *          HTML문서의 모든 요소를 선택한다.
                   <html>태그를 포함한 모든 요소를 선택한다

    + 태그 선택자
        패턴        설명
        태그명      지정된 태그명을 가지는 요소를 선택한다.
                    같은 태그명을 가진 모든 요소를 선택한다.

        예시)
            p{color:red;}

    + 클래스 선택자
        패턴          설명
        .class속성값    class속성값을 지정하여 일치하는 요소를 선택한다.
                        clsss속성값에 지정된 값을 가지는 모든 요소를 선택한다.

        예시)
            /* class 속성값에 container를 포함하고 있는 모든 요소를 선태그 */ 
            .container {color:red;}


    + 아이디 선택자
        패턴            설명      
        #id속성값       id속성값을 지정하여 일치하는 요소를 선택한다.
                        id속성값은 중복될 수 없는 유일한 값이므로 요소가 하나만 선택된다.
        
        예시)
            #image-box {border:1px solid #ddd;}

    + 속성 선택자
        패턴                        설명
        선택자[속성]                지정된 속성을 갖는 모든 요소를 선택한다.
        선택자[속성="속성값"]        지정된 속성값과 일치하는 값을 가지는 모든요소를 선택한다.
        선택자[속성^="속성값"]       지정된 속성값으로 시작하는 모든 요소를 선택한다.
        선택자[속성$="속성값"]       지정된 속성값으로 끝나는 모든 요소를 선택한다.
        선택자[속성*="속성값"]       지정된 속성값을 포함하고 있는 모든 요소를 선택한다.
        선택자[속성~="속성값"]       지정된 속성값을 공백으로 구분된 단어로 포함하는 모든 요소를 선택한다.

        예시)
            a[herf]                 {color:red;}
            a[target="_blank"]      {color:red;}
            a[href^="http]          {color:red;}
            a[href$=".html"]        {color:red;}
            a[href*="공원"]          {color:red;}
            a[href~="test"]         {color:red;}

            .image-item[alt="그림"] {display:none;}
            <img  src="sample.png" class="image-item" alt="그림">
            
    + 복합 선택자
        2개 이상의 선택자를 이용해서 요소를 선택하는 것

        후손(하위요소) 선택자
            패턴                설명
            선택자1 선택자2        선택자1의 후손 중에서 선택자2와 일치하는 요소를 선택한다.
                                  후손은 자신보다 n level 하위에 속하는 모든 요소다.
        

            예시)
                div p                               {color:red;}
                #image-box img                      {display:none;}
                #loginform input[type="text"]       {border : 1px solid red;}
        자손(자식요소) 선택자
            패턴                설명
             선택자1 > 선택자2      선택자1의 자식 중에서 선택자2와 일치하는 요소를 선택한다.
                                   자식은 자신보다 1 level 하위에 속하는 모든 요소다.

            
            예시)
                container > p               {color: red;}
                #image-box > button         {display:none;}
                .naivation > a              {bakcground-color: #ddd;}

        인접 형제 선택자
            패턴                    설명
             선택자1 + 선택자2      선택자1의 형제 중에서 선택자1의 바로 뒤에 위치하는 선택자2 요소를 모두 선택한다.   

             예시)
                #heading + p                 {color:red;}


        일반 형제 선택자
            패턴                    설명
             선택자1 ~ 선택자2      선택자1의 형제 중에서 선택자1의 뒤에 위치하는 선택자2 요소를 모두 선택한다.
                #heading ~ p                   {color:red;}

 div > h3+p
    + 가상 클래스 선택자
        구조 가상 클래스 선택자
            패턴                        설명
            선택자:first-child         선택자로 선택된 요소 중에서 모든 첫번째 자식인 요소를 선택한다.
            선택자:last-child          선택자로 선택된 요소 중에서 모든 마지막번째 자식인 요소를 선택한다.
            선택자:nth-child(n)        선택자로 선택된 요소 중에서 모든 n번째 자식인 요소를 선택한다.

        가상 링크 선택자, 가상 동적 선택자
            패턴                    설명
            a:Link                  a태그 중에서 방문하지 않은 링크를 선택한다.
            a:visited               a태그 중에서 방문한 링크를 선택한다.
            선택자:hover            선택자에 해당하는 요소 중에서 마우스가 올라와 있는 요소를 선택한다.
            선택자:active           선택자에 해당하는 요소 중에서 클릭한 요소를 선택한다.
            선택자:focus            선택자에 해당하는 요수 중에서 포커스가 들어와 있는 요소를 선택한다.

        가상 UI요소 상태 선택자
            패턴                        설명
            선택자:checked              선택자로 선택된 체크박스/라디오버튼 중에서 체크된 요소를 선택한다.
            선택자:enabled              선택자로 선택된 폼입력양식 요소 중에서 사용가능한 요소를 선택한다.
            선택자:disabled             선택자로 선택된 폼입력양식 요소 중에서 비활성화된 요소를 선택한다.
